# 🚀 Intermediate HTML, CSS & JavaScript

A modern front-end web development project showcasing responsive design, interactive user interfaces, and JavaScript-powered functionality. This project demonstrates essential concepts of HTML, CSS, and DOM manipulation while following responsive web design principles.

## ✨ Highlights
- Responsive and modern UI
- Interactive contact form with validation
- Dynamic JavaScript functionality
- Mobile-friendly design using Flexbox and CSS Grid
- Clean, maintainable, and reusable code

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 Objective
To strengthen front-end development skills by building responsive, interactive, and user-centric web applications using modern web technologies.
